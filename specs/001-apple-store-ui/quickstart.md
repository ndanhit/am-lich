# Quickstart: Apple Store UI/UX Redesign

**Feature**: 001-apple-store-ui

## Overview

This feature applies Apple-style UI/UX principles (large typography, card-based layouts, and smooth animations) to the existing vanilla TypeScript lunar calendar web app.

## Development Loop

1. **Install dependencies** (if not already):
   ```bash
   npm install
   ```

2. **Start the development server** (builds UI and serves locally):
   ```bash
   npm run dev
   ```

3. **Verify Design Changes**:
   - Open the provided local URL (typically `http://localhost:3456`).
   - Navigate between the Calendar and Upcoming tabs.
   - Verify that fonts scale appropriately, cards display with proper rounded corners & shadows, and interactions (hover, click, modal open) feel smooth and Apple-like.
