# Quickstart: UI/UX Refinement & Bug Fixes

## Navigation
Month navigation is now automatic via vertical scroll.
- **Desktop**: Use mouse wheel or trackpad scroll up/down.
- **Mobile**: Swipe up/down on the calendar grid area.

## Floating "Today" Button
If you navigate away from the current month, a floating button with the label **"Hôm nay"** will appear at the bottom right. Click it to return home.

## Calendar Grid
The grid now starts on **Monday**. Sundays are highlighted in red for better visibility of weekends.

## Mobile Fonts
Calendar numbers are automatically scaled up by 20% on devices with screen widths less than 768px.
